# Contributing to Kitchen Sync

## Getting Started

1. Clone the repository.
2. Read `docs/requirements/` for product context.

## AI Agents

AI agents contributing to this repository must follow the guidelines in `.agent/AGENTS.md`. This includes:
- Always working on feature branches (never push to `main` directly)
- Opening Pull Requests for all changes
- Using conventional commit messages

## Pull Requests

- Create a new branch for your feature or fix.
- Provide a clear description of changes.
- See `.agent/workflows/create-pr.md` for the standardized PR workflow.

## Task Management

We use `bd` (Beads) for task tracking. Run `bd help` for commands.
