# Meal Planning

Flexible weekly meal scheduling with recipe selection and ratings.

## Committed

- "Week" is a flexible, preference-based concept (e.g., 4 meals Mon–Thu at 6pm)
- Default plan duration is one week
- Default is one meal per day, but multiple meals per day are supported
- Single recipe per meal slot (no multi-recipe meals)
- Users set default scheduling preferences that can change over time
- One-time overrides are always possible (e.g., skip Tuesday this week)
- Select recipes for upcoming meals
- Any household member can publish a meal plan (no approval required)
- Per-person ratings: like vs. don't like, with household aggregate view
